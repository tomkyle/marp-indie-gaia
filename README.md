### Development Notes

Add Git subtree:

```shell
$ git subtree add --prefix=themes git@github.com:marp-team/marp-core.git main --squash
```

Update subtree:

```shell
$ git subtree pull --prefix=themes git@github.com:marp-team/marp-core.git main --squash
```
